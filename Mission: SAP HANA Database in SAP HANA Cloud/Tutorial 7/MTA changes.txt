Open the mta.yaml file and add the following syntax on top of the Resources:

- name: NAME OF YOUR USER PROVIDED SERVICE

Still on the mta.yaml file, add also the following syntax at the end of the file:
- name: NAME OF YOUR USER PROVIDED SERVICE
  type: org.cloudfoundry.existing-service

Example of working mta.yaml file with correct indentation:

(_schema-version: "2.1"
ID: BRT_Data
version: 0.0.1
modules:
- name: db
  type: hdb
  path: db
  requires:
  - name: hdi_db
    properties:
      TARGET_CONTAINER: ~{hdi-container-name}
  - name: MYBRT
resources:
- name: hdi_db
  type: com.sap.xs.hdi-container
  properties:
    hdi-container-name: ${service-name}
- name: MYBRT
  type: org.cloudfoundry.existing-service
